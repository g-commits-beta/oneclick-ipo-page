「パスキーを設定してください」

最近、証券会社にログインするとこんなメッセージが表示されることが増えていませんか？ SBI証券、マネックス証券、楽天証券、SMBC日興証券、野村證券… 主要な証券会社が次々とパスキー認証を導入しています。

でも「パスキーって何？」「どうやって設定するの？」と戸惑っている方も多いはず。この記事では、パスキーの仕組みと**最も簡単に始める方法**を解説します。

## パスキーとは？

パスキーは、**パスワードに代わる新しい認証方式**です。

従来のパスワード認証では、あなたが覚えているパスワードをサーバーに送信して照合していました。パスキーは、あなたのデバイスに保存された「秘密鍵」を使って認証を行います。パスワードのように盗まれたり、使い回しで漏洩するリスクがありません。

| | パスワード | パスキー |
|------|----------|---------|
| 認証方法 | 文字列を入力 | 生体認証やPINで確認 |
| 保存場所 | サーバーとあなたの記憶 | あなたのデバイスのみ |
| フィッシング | 騙されて入力するリスクあり | **原理的に不可能** |
| 使い回し | ついやりがち | 各サイトごとに自動生成 |
| 入力の手間 | 毎回入力 | ワンタッチ |

特に証券会社のように**お金を扱うサービスでは、パスキーのセキュリティ向上は大きなメリット**です。フィッシング詐欺による不正ログインが原理的に防げるため、各社が積極的に導入を進めています。

## パスキーを管理するツール

パスキーをどこに保存するかが重要です。主な選択肢は3つあります。

### 1. Bitwarden（おすすめ）

**無料で使えるオープンソースのパスワードマネージャー**です。

- **料金**: 基本機能は**完全無料**（有料プランは年間$10）
- **セキュリティ**: ゼロナレッジ暗号化を採用。あなたのデータは暗号化された状態でしか保存されず、**Bitwarden運営側でも中身を閲覧できません**。さらにオープンソースなので、コードが公開されており第三者による監査も行われています
- **対応環境**: Windows / Mac / Linux / iOS / Android / 全主要ブラウザの拡張機能
- **パスキー管理**: ブラウザ拡張から直接パスキーの作成・使用が可能
- **デバイス間同期**: スマホで作成したパスキーをPCでも使える

Bitwardenの最大の強みは、**無料でここまでの機能とセキュリティを提供している**点です。パスワードの保存数に制限もなく、複数デバイス間の同期も無料プランで利用可能。証券口座のように重要な認証情報を管理するのに最適です。

### 2. Windows Hello

Windowsに標準搭載されたパスキー管理機能です。

- **料金**: 無料（OS標準機能）
- **認証方法**: 指紋認証・顔認証（Windows Hello対応カメラ）・PIN
- **メリット**: 追加インストール不要ですぐ使える
- **デメリット**: **そのPCでしか使えない**。PC買い替え時にパスキーの移行が困難。スマホとの同期もできない

手軽に始められる反面、複数PCや複数デバイスを使う方には不向きです。「とりあえず1台のPCで試してみたい」という方には良い選択肢。

### 3. iCloudキーチェーン

Apple製品に標準搭載されたパスキー管理機能です。

- **料金**: 無料（Apple ID標準機能）
- **認証方法**: Face ID / Touch ID
- **メリット**: iPhone・iPad・Mac間で**自動同期**。設定不要で使える
- **デメリット**: **Apple製品限定**。Windowsでは基本的に使えない（iCloud for Windows経由で一部対応）

Apple製品で統一している方には最も手軽な選択肢ですが、IPO投資ではWindows PCを使う場面も多いため、制約になる場合があります。

### 4. Google パスワードマネージャー

Google Chromeに内蔵されたパスキー管理機能です。

- **料金**: 無料
- **認証方法**: Googleアカウント + デバイスの画面ロック
- **メリット**: Chromeユーザーなら追加設定なしで利用可能。Android端末間で自動同期
- **デメリット**: **Chrome以外のブラウザでは使えない**。Googleアカウントに依存するため、アカウント停止時のリスクがある

普段からChromeを使っている方には手軽ですが、ブラウザを限定される点に注意。

### 5. 1Password

定番の有料パスワードマネージャーです。

- **料金**: **月額約400円〜**（年払い$36〜）
- **セキュリティ**: シークレットキー + マスターパスワードの二重保護
- **対応環境**: 全OS・全ブラウザ対応
- **メリット**: UIが洗練されていて使いやすい。家族プランあり。企業での導入実績が豊富
- **デメリット**: **有料**。無料で同等機能のBitwardenがあるため、コスト面で不利

高品質なUIとサポートが特徴ですが、個人利用であれば無料のBitwardenで十分です。

### 6. Dashlane

海外で人気の高い有料パスワードマネージャーです。

- **料金**: 無料プランあり（25件まで）/ 有料は月額約500円〜
- **特徴**: VPN機能付き、ダークウェブモニタリングあり
- **デメリット**: 無料プランの制限が厳しい。日本語対応がやや弱い

機能は豊富ですが、証券口座のパスキー管理だけが目的なら過剰スペックです。

## Bitwardenでパスキーを設定する手順

最もおすすめの**Bitwarden**を使って、証券会社のパスキーを設定する手順を解説します。

### ステップ1: Bitwardenをインストール

1. [Bitwarden公式サイト](https://bitwarden.com)でアカウントを作成（無料）
2. 使っているブラウザ（Chrome / Edge / Firefox）の**拡張機能**をインストール
3. 拡張機能にログイン

### ステップ2: 証券会社でパスキーを登録

SBI証券の場合:

1. SBI証券にログイン
2. 「設定」→「セキュリティ設定」→「パスキー設定」
3. 「パスキーを登録」をクリック
4. Bitwardenのポップアップが表示される → 「保存」をクリック
5. 完了！

他の証券会社でもほぼ同じ流れです。各社の設定画面で「パスキー」や「生体認証」のメニューを探してください。

### ステップ3: パスキーでログイン

1. 証券会社のログインページを開く
2. 「パスキーでログイン」ボタンをクリック
3. Bitwardenのポップアップが表示される → 確認するだけ
4. ログイン完了

パスワードを入力する必要はありません。**ワンクリックでログイン**できます。

## 複数口座をパスキーで管理するメリット

IPO投資では当選確率を上げるために**7〜9社の証券口座**を持つのが一般的です。毎回パスワードを入力してログインするのは大きな手間です。

Bitwardenでパスキーを管理すれば:

- **全口座のパスキーを一元管理**
- **ログインがワンクリック**に
- **パスワード漏洩・フィッシングのリスクをゼロに**
- **PC買い替え時もBitwardenにログインするだけで移行完了**

## パスキーの注意点

### マスターパスワードは絶対に忘れない

Bitwardenのマスターパスワードを忘れると、保存した全パスキーにアクセスできなくなります。安全な場所にメモしておきましょう。

### ブラウザ拡張が必要

パスキーの作成・使用にはBitwardenのブラウザ拡張機能が必要です。アプリ単体では動作しません。

### 対応していない証券会社もある

大和証券・松井証券は2026年現在、パスキーの完全対応がまだです。これらの証券会社では従来のパスワード認証を使う必要があります。

## まとめ

パスキーは「難しそう」に見えますが、実際にやることは:

1. **Bitwardenをインストール**（5分）
2. **各証券会社でパスキーを登録**（1社あたり2分）
3. **以降はワンクリックでログイン**

一度設定してしまえば、毎回のログインが劇的に楽になります。IPO投資で複数の証券口座を使っている方は、ぜひパスキー + Bitwardenの組み合わせを試してみてください。

## ワンクリIPOはBitwarden連携に対応

ワンクリIPOでは、Bitwardenと連携して各証券会社へのログインを効率化しています。パスキー認証を活用することで、7社の証券口座への申し込み管理をスムーズに行えます。
